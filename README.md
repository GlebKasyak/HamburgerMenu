#  HamburgerMenu

## Описание приложения

Скрывающееся мобильное меню.

## Установка приложения

Для запуска приложения сначала клонируйте репозиторий

```sh
git clone https://github.com/GlebKasyak/HamburgerMenu.git
```

Установите все зависимости

```sh
npm install
```

### Запуск приложения

```sh
npm start
```
